Abalabos: The Ascent of the Black Wizard

(this article is inprogress of writing)

Details:

  • A 3D game inspired by Celeste.
  • Game project 3.
  • 12-week development (20 hours per week) with a wonderful team of 5 programmers, 2 level designers and 4 artists.
  • Game is made with TGA's inhouse engine (TGE) in C++ and Unity as Editor
  • https://carry.itch.io/abalabos 

 

My contributions:

  • Simple and short tutorial videos for my team how to setup and install Perforce, Unity and Visual Studio etc
  • Export and Import data (gameobjects, ui, scenes etc) from Unity to TGE
  • SceneManager & GameObject System
  • Rendering pipeline
  • Object Culling
  • Thread loading data and cache
